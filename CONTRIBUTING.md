By submitting a contribution to this repository, you agree that your
contribution is licensed under the **Apache License, Version 2.0**, the same
license as [Ghidra](https://github.com/NationalSecurityAgency/Ghidra).

See [LICENSE](LICENSE) and [NOTICE](NOTICE).

## Guidelines

- Do not commit API keys, tokens, or other secrets.
- New bundled third-party code must be reflected in `NOTICE` (and `docs/LICENSES.md` when needed).
- Prefer stable `ghidra.vibe.*` accessibility identifiers for new UI controls.
- Keep macOS Liquid Glass usage on official APIs only (see `docs/macos-liquid-glass.md`).
